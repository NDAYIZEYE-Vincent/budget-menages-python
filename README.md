# Analyse des dépenses des ménages avec Python

## Objectif
Ce projet vise à analyser la structure des dépenses des ménages afin
d’identifier les facteurs de vulnérabilité financière et proposer
des recommandations pour améliorer le bien-être économique.

## Données
Les données sont simulées et représentent 100 ménages avec :
- Revenu mensuel
- Taille du ménage
- Dépenses par catégorie (alimentation, logement, santé, etc.)

## Méthodologie
- Nettoyage et préparation des données
- Analyse exploratoire (EDA)
- Visualisation des tendances
- Identification des ménages vulnérables

## Résultats clés
- L’alimentation et le logement représentent la plus grande part des dépenses
- Une proportion importante des ménages est en déficit budgétaire
- Les ménages de grande taille ont un revenu par personne plus faible

## Recommandations
- Cibler les ménages vulnérables pour les programmes d’assistance
- Promouvoir la planification budgétaire
- Orienter les politiques sociales vers la réduction des dépenses essentielles

## Outils utilisés
- Python
- Pandas
- Matplotlib
- Jupyter Notebook
